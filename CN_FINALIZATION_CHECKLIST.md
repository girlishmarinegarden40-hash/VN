# 爱丽丝庭院中文定稿清单

## 1. 当前结论

- 当前主线已经完整，`可以进入中文定稿阶段`。
- 现在不建议继续横向扩写大设定、主角色、宇宙层说明。
- 现在最该做的是：`收束、统一、润到稳`，然后进入英文版。

一句话判断：

`剧情已经够了，后面主要做提纯，不做扩张。`

## 2. 总体原则

- 不新增主角色。
- 不新增大型世界观层级。
- 不把作者秘密来源写进正文。
- 不把后期哲学结论解释得更明。
- 不为了“更完整”去破坏现在已经成立的留白。

## 3. 当前最适合锁定不动的部分

这些部分已经成立，原则上不要大改：

- `A-017 -> A-403 -> GOD_KILL -> After Alice -> Noa -> 第十三张空白牌` 这条主线。
- `A-017 把水母馆传单交给 Noa` 这个场面。
- `Visitor` 作为外部观看和贴标签机制的进入方式。
- `A-403` 不是反派，而是系统逻辑反证的定位。
- `After Alice` 不是重启，而是未来并存的状态。
- `Noa` 不是补偿，不是谁的替代品。
- `Sienna / Moon / Dea` 作为更后来的轻层余韵。

## 4. 只建议微调的地方

### 4.1 A-301

- 现在 `A-301` 在正文中被提到多次，但没有像 `A-092 / A-214` 那样获得读者可感知的完整形象。
- 这不是主线漏洞，但如果你心里一直觉得有一点空，这会是最值得补的一个点。
- 建议：如果补，只补`一次明确且短的个案存在感`，不要再开一整卷。

### 4.2 早期章节的进入速度

- 开篇气质已经很好。
- 但英译时，前段若太连续地堆意象，英文读者会更容易“敬佩氛围，但还没抓住线”。
- 中文定稿时，可轻微检查前 4 章：
  - 是否每章都能让读者抓住一个明确推进点
  - 是否偶尔有句子太美但推进太少

不是删美感，而是确保“美”和“推进”一起在。

### 4.3 间章密度

- 间章整体是有价值的，不建议现在删。
- 但中文定稿时可以统一检查：
  - 有没有两篇间章在做同一件事
  - 有没有一两篇更适合缩短而不是重写
  - 有没有篇名和内容重心略有偏差

目标不是减量，而是让每一篇都更“必要”。

### 4.4 后段关键词重复

- 后段已经非常成熟，但也开始密集出现这些关键词：
  - 被看见
  - 记录
  - 判定
  - 答案
  - 未来
  - 留白
- 这些词是核心，但中文定稿时要检查：
  - 有没有几处其实在说同一件事
  - 有没有某一章里一个词出现过密

原则：

`核心词要反复回归，但每次回归都最好有一点新变义。`

## 5. 章节层面的定稿建议

### 第一段：进入庭院

涉及：

- `00-prologue`
- `01-touchless-street`
- `02-a017-arrival`
- `03-a017-relive`
- `04-white-flowers-and-undefined`

判断：

- 已经够。
- 不需要再加大设定说明。
- 只需要检查早期句子是否有个别过密处，保证初入读者能抓住：
  - 世界异常
  - A-017 的痛
  - 三姐妹的功能差异
  - “未定义”的重量

### 第二段：庭院与四人偶

涉及：

- `05-all-nameplates-empty`
- `06-a092`
- `07-a214`
- `07-azalea`
- `08-meiling`
- `09-shella`
- `10-mizuki`
- `11-rabbit-ears`

判断：

- 已经形成完整中段。
- 四人偶各自的气质已经成立。
- 现在主要检查的是：四人偶之间有没有谁明显比别人更“像设定”、更少“像人”。

目前最建议重点守住：

- `Meiling` 的“轻轻漏掉”
- `Mizuki` 的“如果我早一点”
- `Shella` 的“替缺席者留位”
- `Azalea` 的“失而不可复得后的确认”

### 第三段：崩裂与反证

涉及：

- `12-visitor`
- `13-a403`
- `14-failed-answer`
- `15-god-kill`

判断：

- 这是目前全作最稳的一段之一。
- 主干已经完整，不建议大改。
- 只建议检查：
  - `Visitor` 的语言是否始终足够轻率、快、伤人
  - `A-403` 的语言是否始终有骨头，而不是沦为单纯“金句机”

### 第四段：After Alice 之后

涉及：

- `16-after-alice`
- `17-noa`
- `18-thirteenth-card`

判断：

- 已经足够作为正式收束。
- 不需要再补“大结局解释”。
- 不要为了让读者更懂，而去讲破 `Noa / Sienna / Moon / Dea`。

最该守住的是：

- `After Alice` 的安静
- `Noa` 的轻
- 第十三张牌的留白感

## 6. 中文定稿时最该逐章检查的项目

每一章都可以按这 6 条快速过一遍：

1. 这一章最清楚的推进点是什么？
2. 这一章最重的意象是什么？
3. 这一章有没有说破本来应该留给读者自己感觉的东西？
4. 这一章里核心词有没有重复过密？
5. 这一章的结尾有没有真正把人推向下一章？
6. 这一章里角色说话，有没有谁开始说得不像她自己？

## 7. 进入英文版前必须先锁的东西

- 中文章节顺序
- 中文正文定稿文本
- 术语口径
- 标题口径
- 是否保留全部间章
- 哪些地方属于“作者心脏”，只留后台，不进正文说明

## 8. 当前建议路线

### 路线 A：最稳

1. 做一轮中文定稿
2. 冻结主线文本
3. 开始英文版

### 路线 B：只补一个点再定稿

1. 如果你始终放不下 `A-301`
2. 只补这一个点
3. 立即冻结正文
4. 开始英文版

## 9. 当前不建议做的事

- 不建议再加大段番外去解释世界真相。
- 不建议再给三姐妹补大量前史说明。
- 不建议再给 `Noa` 增加明确来源解释。
- 不建议把 `WORLD4` 那层现实原型写进正文。
- 不建议把作品后半变成理念辩论文。

## 10. 一句最后提醒

现在这部作品最需要的，不是更多内容。

而是：

`让已经存在的那些句子、那些关系、那些场面，稳定下来。`
